◇推奨モーフ
　・顔修正（最重要）
	ver1.0で指摘が多かった、いわゆる横顔ペラペラ問題を修正するモーフです。
	これを適用することで鼻が高くなり、唇がぷっくりします。
　・こっちみんな
	目を閉じた時に視線が大きく動いていると瞳が目蓋を貫通する場合があります。
	その際はこっちみんなモーフを適用するか、視線をマイルドな感じに調整して下さい。
　・口奥
	二次元特有の口が奥まった表現が可能です。
	真横から見た場合のみ少し不自然な感じになるので、その時だけモーフのスライダーを下げるといい感じになります。
	また、口角奥モーフと併用することで見栄えがより良くなります。
